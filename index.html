O Bê-á-bá de HTML

HTML (Linguagem de Marcação de HiperTexto) é código utilizado para estruturar uma página Web e seu conteúdo e é sobre ela que vamos falar, mas vale lembrar que o objetivo desta página é introduzi-lo no fascinante mundo do desenvolvimento Web e não capacitá-lo para desenvolver um site completo. Entendendo Client x Server

(Voltar)

Antes de começar a aprender HTML, é fundamental darmos uma breve introdução aos principais conceitos envolvidos no desenvolvimento Web, com enfoque na definição de Client e Server

Client

Ao nos referirmos ao lado cliente, estamos falando sobre o lado que consome conteúdo na internet. Conteúdo esse disponibilizado por meio de arquivos estáticos (HTML, CSS, JAVASCRIPT, Imagem, Texto, ...).

Server

O lado servidor, como o nome sugere, fornece informações na internet, é lá onde os sites são hospedados, e as linguagens utilizadas são C#, JAVASCRIPT, Pythom, Ruby, entre outos.

Aplicações Web

Aplicações Web são soluções criadas que possuem a internet como meio de comunicação Client x Server não sendo necessária sua instalação no dispositivo.

Introdução ao HTML na Prática

(Voltar)

Ao se deparar com algum código em HTML, você com certeza visualizou as chamadas TAGs, mesmo que na ocasião não soubesse o que são elas. Sem problemas quanto isso porque normalmente esse é um dos primeiros tópicos dos cursos on-line sobre HTML, onde você aprenderá o que as TAGs são e como utilizá-las

As TAGs

As TAGs vão ser as responsaveis por definir a estrutura do nosso site. Entre elas, encontra-se, por exemplo, a TAG h1, que define como título aquilo que é escrito entre seu abrimento e fechamento. Assim como h3, temos várias outras TAGs interessantes que lhe ajudará muito na estruturação do seu site, as quais, em geral, são veementemente exploradas no deocorrer dos cursos sobre HTML

Os atributos

Os atributos ficam nas TAGs de abertura e objetivam controlar o comportamento da TAG em questão.

Nessa parte do aprendizado, podemos exemplificar com o atributo id, o qual nomeia uma TAG e é de fundamental importância para o desenvolvimento de aplicações, uma vez que pode servir para identificar um componente do site para uma possível modificação, ação essa banal na maioria dos projetos de programação Web. Além dessa, existem vários outros atributos que lhe permitirão variar a estrutura dos componentes do seu site.
